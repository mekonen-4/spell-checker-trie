# Mekelle University - DSA Project (Group 4)
## High-Performance Spell Checker using Trie Data Structure

### Project Overview
An efficient spell-checking application built with the MERN stack. It utilizes a **Trie (Prefix Tree)** for $O(L)$ search complexity and **Edit Distance (d=1)** for suggestions.

### How to Run
1. **Dictionary:** Download `words_dictionary.json` and place it in `backend/data/`.
2. **Backend:** - `cd backend`
   - `npm install`
   - `npm run dev`
3. **Frontend:** - `cd frontend`
   - `npm install`
   - `npm run dev`
